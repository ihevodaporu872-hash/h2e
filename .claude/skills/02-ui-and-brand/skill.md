# Skill: UI & Brand Quality (Premium look)

## Цель
Сделать визуал уровня профессиональной системы: чистые сетки, продуманные отступы, уверенная композиция, продуманные состояния.

## Бренд-акценты (позиционирование H2E)
- Система управления строительной документацией
- "Инженерная логика", "прозрачность", "контроль"
- Чёткие этапы работы, статусы документов, метрики проекта

## Правила UI
- Responsive сетка (CSS Grid / Flexbox)
- Единый контейнер и вертикальный ритм
- CSS Modules для изоляции стилей
- CSS Variables для темизации

## Типографика
- Заголовки: 2–3 уровня
- Body: читабельный, межстрочный интервал
- Длинные тексты дробить: списки, подзаголовки

## Цветовая схема
- Используй CSS Variables: `--text-primary`, `--bg-primary`, `--border-color`
- Поддержка темной темы через ThemeContext
- Акцентные цвета для статусов: blue, green, yellow, red

## Интерактив
- Все кнопки/ссылки имеют hover/focus
- Loading states для асинхронных операций
- Понятные состояния: draft, review, approved, rejected

## Иконки
- Используй lucide-react
- Размеры консистентны (16, 18, 20, 24)
- Иконки дополняют текст, не заменяют

## DoD
- Страница выглядит профессионально
- CTA заметны, но не кричат
- Нет визуального шума
- Темная тема работает корректно
